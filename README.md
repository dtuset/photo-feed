# photo-feed
Image Gallery
## Project setup
```
npm install
```
### Go to **./photo-feed/appConfig.js** and replace **XXXXX** with an Api Key
```
export default {
  clientKey: { apiKey: 'XXXXX' },
};
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
